# Usage

## 导入与书架

在主界面：

- 点击 **导入 txt** 选择文件
- 导入后会出现在 **书架**
- 点击一本书，在右侧查看 **目录与控制**

## 目录分页

目录默认 **20 章/页**：

- 支持 **上一页/下一页**
- 支持下拉选择页码区间（`n×20` 的区间）
- “从本章阅读”按钮为 hover 悬浮，不占位（避免右侧留白）

## 阅读条（Overlay）

- **点击正文**：显示工具栏
- **拖动正文**：移动阅读条窗口
- **拖动边缘/角**：缩放阅读条窗口

## 工具栏

工具栏按钮包括：

- 开始/暂停
- 上一章/下一章
- 上一页/下一页
- 字号减小/增大
- 设置
- 关闭

按钮显示可在设置窗中勾选配置（默认全选）。

## 快捷键

- 显示/隐藏阅读条：`Ctrl` + `Shift` + `X`（macOS 为 `⌘` + `Shift` + `X`）
