## 过程备注
- 以“官方源码/官方文档”为主，优先规避二手博客结论。
- 对“single line reading group”关键词，开源资料更偏“展示层工具”；其正文提取通常复用 reader mode 思路或简单清洗，不是独立高鲁棒抽取引擎。
- WebToEpub 的优势不在单一算法，而在“海量站点 parser + default parser + 人工可配置 selector”的组合兜底。
